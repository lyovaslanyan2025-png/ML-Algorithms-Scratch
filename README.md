# Machine Learning Algorithms From Scratch

This repository contains machine learning algorithms implemented from scratch using Python and NumPy.

## Goals

- Understand the mathematics behind ML algorithms
- Implement algorithms without using scikit-learn
- Compare implementations with scikit-learn
- Improve Python programming skills

## Algorithms

- [ ] Linear Regression
- [ ] Logistic Regression
- [ ] Decision Tree
- [ ] Random Forest
- [ ] K-Nearest Neighbors
- [ ] Naive Bayes
- [ ] Support Vector Machine
- [ ] K-Means
- [ ] PCA

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
